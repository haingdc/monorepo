### How to run

- Web
  - [CRA] `$ yarn workspace web-cra start`
- Mobile
  - [iOS]
    - `$ cd packages/mobile/ios && pod update && sudo gem install cocoapods`
    - [CLI]
      - `$ yarn workspace mobile start`
    - [Xcode]
      - Press the Run button
